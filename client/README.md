# Collaborative Editor

A real-time collaborative editor built using React.js, Node.js, Express.js and Socket.IO.

## Features
- Real-time collaborative editing
- Multiple users support
- Modern responsive UI
- Live synchronization

## Tech Stack
- React.js
- Tailwind CSS
- Node.js
- Express.js
- Socket.IO

## Run Locally

Frontend:
npm install
npm run dev

Backend:
npm install
node server.js
